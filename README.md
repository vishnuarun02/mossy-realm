# MossyRealm

A tiny moss-covered corner of the internet. A forest realm stitched together by curiosity and wandering.

## About

MossyRealm is a personal website built with a 90s/early 2000s Neocities aesthetic. It features a cozy, whimsical design inspired by old-school personal homepages.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** TailwindCSS
- **Fonts:** Google Fonts (Cinzel, Lora, Cormorant, Mystery Quest)

## Project Structure

```
mossyrealm/
├── mossy-realm/          # Next.js application
│   ├── app/              # App router pages & layouts
│   ├── components/       # React components
│   └── public/           # Static assets
└── design-kitchen/       # Design references & documentation
```

## Getting Started

```bash
cd mossy-realm
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Design

- **Color Palette:** Sunlit Glade (warm forest greens & golden accents)
- **Typography:** Enchanted Forest Storybook font stack
- **Layout:** Centered three-column grid with fixed wallpaper background

See `/design-kitchen` for detailed design documentation.
